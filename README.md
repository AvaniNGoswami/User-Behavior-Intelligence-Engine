# User Behavior Intelligence Engine

End-to-end pipeline to simulate, clean, sessionize, and analyze user behavioral data for actionable insights.

## What it does
- Simulates real-world user event data with noise
- Handles missing values, duplicates, and inconsistencies
- Implements sessionization based on inactivity thresholds
- Engineers behavioral features from session data
- Segments users using clustering techniques
- Uses LLMs to generate behavioral insights from cluster data
  
## Why this is different
Focuses on real-world messy data and behavioral patterns instead of clean datasets.

## Tech stack
Python, Pandas, NumPy, Scikit-learn
